# Technical Task Generator Skill

## Description
Bu skill mürəkkəb biznes istəklərini təmiz və professional proqram təminatı tapşırıqlarına (Technical Specifications) çevirmək üçün istifadə olunur.

## Qaydalar
1. **Dəqiqlik**: Bütün dəyişənlər və yollar (`absolute paths`) qeyd olunmalıdır.
2. **Texnologiya**: Mövcud stack-ə (React, Vite, Supabase, Tailwind) uyğunluq.
3. **Müstəqillik**: Yaradılan tapşırıq istənilən proqramçı tərəfindən başa düşülən olmalıdır.

## Nümunə Format
### Objective
[Brief description]

### Changes Required
- **Frontend**: [Files to modify/create]
- **Backend/Supabase**: [Migrations/Policies]
- **API**: [New endpoints or hooks]

### Security & Performance
[Considerations]
